🚀 Quick Start (No Setup Required!)

  1. Make sure your Google Sheet is accessible (shared with "Anyone with the link can edit")
  2. Open the extension popup and go to "Settings" tab
  3. The configuration should already be filled in and enabled
  4. Go to any LinkedIn profile and click "Extract Profile Data"
  5. Check your Google Sheet - data should appear automatically!

Step 1: Get Google Sheets API Key

  1. Go to the Google Cloud Console
  2. Create a new project or select an existing one
  3. Enable the Google Sheets API:
    • Go to "APIs & Services" → "Library"
    • Search for "Google Sheets API"
    • Click on it and press "Enable"
  4. Create credentials:
    • Go to "APIs & Services" → "Credentials"
    • Click "Create Credentials" → "API Key"
    • Copy the generated API key
  5. Configure API key restrictions (recommended):
    • Click on your API key to edit it
    • Under "API restrictions", select "Restrict key"
    • Choose "Google Sheets API" from the dropdown
    • Save the changes

Step 2: Configure Google Sheets Permissions

  1. Open your Google Sheets document
  2. Click the "Share" button in the top right
  3. Change the access to "Anyone with the link can edit" or "Anyone with the link can view"
  4. Copy the sharing link (you already have this)
Important: The sheet must be publicly accessible for the API key to work. If you want to keep it private, you'll need to use OAuth2 authentication instead.

Step 3: Configure the Extension

  1. Open the LinkedIn Agent extension popup
  2. Click on the "Settings" tab
  3. Enable "Google Sheets Integration"
  4. Enter the following configuration:
    • API Key: Your Google Sheets API key from Step 1
    • Spreadsheet ID: YOUR_GOOGLE_SHEET_ID
    • Sheet Name: Sheet1 (or your preferred sheet name)
  5. Click "Save Configuration"

Step 4: Test the Integration

  1. Navigate to any LinkedIn profile page
  2. Click the extension icon
  3. Click "Extract Profile Data"
  4. Check your Google Sheets document for the extracted data

🔧 Troubleshooting

❌ Common Issues

  • API Key Error: Make sure your API key is valid and has access to Google Sheets API
  • Permission Error: Ensure the Google Sheet is publicly accessible
  • Sheet Not Found: Check that the sheet name matches exactly
  • Quota Exceeded: Google Sheets API has daily limits

✅ Success Indicators

  • Extension popup shows "📊 Google Sheets: Data exported automatically"
  • New rows appear in your Google Sheet after extraction
  • Browser console shows "✅ Successfully sent data to Google Sheets"

🔒 Security Notes

  • Keep your API key secure and never share it publicly
  • Consider using IP restrictions on your API key
  • Regularly rotate your API keys
  • Monitor your Google Cloud usage for unexpected activity
